## `01 / what I build`

I build backend-heavy products where the interesting part lives between **clean APIs**, **async workflows**, **data**, and an interface people actually enjoy using.

```text
request  ──▶  FastAPI  ──▶  domain logic  ──▶  PostgreSQL
                       └──▶  async events  ──▶  RabbitMQ
                                           └──▶  useful UI
```
